Front-end development vs back-end development, different kinds of coding, what to expect to be able to do at the end of this workshop

vocabulary, terminology?

Front-end development is also called client-side development. Basically this means that it makes what the user sees and interacts with.

Website development is split into two kinds of code - we call it front-end development and back-end development. Front-end development is made up of coding languages such as HTML, CSS, and Javascript, and back-end development has a multitude of languages - the most common are .NET and PHP, but also has other ones that I’m not as familiar with. I’m a front-end developer so that’s going to be the focus of the workshop

Free Learning

  • codeacademy

    Codeacademy is probably one of the best free resource for learning both front end code and back end code.

  • khan academy

    Khan Academy has some good introductory courses for HTML/CSS, as well as some more advanced scripting and database stuff.

Courses

  • lighthouse labs

    Coding boot camp, with instructors and locations across Canada and helps with job placement upon course completion.

Tools + Programs

  • brackets

    Brackets is a code editing program that is easy to setup and run, and you can see your code right away. Also has a preview mode so you can see what you’re doing in the browser.

  • google fonts

    Google Fonts is a great resource for putting custom fonts in your code without having to download them or host them yourself. And they're free!
